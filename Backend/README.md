# OC_P6_AA

## BackEnd
Clone this repo:
``git clone https://github.com/AGPA-88/OC_P6_AA``

In the terminal do:

``cd OC_P6_AA/``

``npm install``

``node server`` or ``npm start``

## FrontEnd
Clone this repo:
``git clone https://github.com/OpenClassrooms-Student-Center/Web-Developer-P6.git``

In the terminal do:

``cd Web-Developer-P6/``

``npm install``

``npm start``
